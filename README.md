# OG Card Maker

🎨 **Free Open Graph / Social Media Image Generator**

Create beautiful social media preview images in seconds. No signup, no watermark, 100% free.

## Features

- ⚡ **Instant Preview** - See changes in real-time
- 🎨 **12+ Themes** - Beautiful gradients & color schemes
- 📐 **Perfect Size** - 1200×630 optimized for all platforms
- 🔤 **Multiple Fonts** - Sans-serif, serif, mono, rounded
- 🌊 **8 Patterns** - Dots, grid, waves, circles & more
- 📱 **3 Layouts** - Centered, left-aligned, bottom-aligned
- 🆓 **100% Free** - No signup, no watermark, forever free
- 🔒 **Privacy First** - Everything runs in your browser

## Use Cases

- Blog post social media cards
- Open source project banners
- Tweet/LinkedIn share images
- Product launch announcements
- YouTube video thumbnails
- Discord embed images

## Tech Stack

- Pure HTML/CSS/JavaScript
- HTML5 Canvas API
- Zero dependencies
- Hosted on Vercel (free tier)

## Development

```bash
npx serve .
```
