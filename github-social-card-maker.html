What is a GitHub Social Card?

A GitHub social card is the preview image that appears when someone shares your repository link on social media (Twitter, Facebook, LinkedIn, Discord, Slack). By default, GitHub auto-generates one, but a custom card makes your project stand out.

How to Set a Custom Social Card on GitHub

  • Create a .github folder in your repository root
  • Add a social-preview.png file (1280×640 pixels)
  • GitHub will automatically use it as the social card
  • Alternatively, go to repo Settings → Social Preview → Upload image

GitHub Social Card Size

  • Recommended: 1280×640 pixels
  • Aspect ratio: 2:1
  • Format: PNG (recommended) or JPG
  • Max file size: 5MB

Tips for Effective GitHub Social Cards

  • Show your repo name prominently
  • Include a one-line description of what it does
  • Use your project's brand colors
  • Add stars/forks badge if impressive
  • Keep it simple — people scroll fast
